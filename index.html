Prototypes and 'This'

There are four types of binding:

  • Window binding

Window Binding

Using the this keyword without any context will default to binding to the Window object. For example, writing this.alert('Hello, friend') will create an alert in the browser.

If you write console.log(this), you will see the Window object in the console with all of the browser's properties such as alert, confirm, cookieStore, document, length, localStorage, onmouseover, etc.

The Window can be thought of as the 'highest' object in JavaScript.
